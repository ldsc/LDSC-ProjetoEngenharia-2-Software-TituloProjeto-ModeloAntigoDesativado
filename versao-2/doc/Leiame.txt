A ideia é que o usuário entre primeiro aqui e tenha acesso a documentação básica de Uso e Desenvolvimento do software;

Arquivos e diretórios:
=======================

ManualDoDesenvolvedor.pdf: 
	Cópia do manual do desenvolvedor (como foi desenvolvido o software).
ManualDoUsuario.pdf: 
	Cópia do manual do usuário (como usar o software).
ManualTecnico.pdf:
	Cópia do manual técnico (detalhes técnicos, equações)	
	
ManualDoDesenvolvedor:
Diretório com todos os arquivos para gerar o manual do desenvolvedor.

ManualDoUsuario:
Diretório com todos os arquivos para gerar o manual do usuário.

ManualTecnico:
Diretório com todos os arquivos para gerar o manual técnico-científico.
